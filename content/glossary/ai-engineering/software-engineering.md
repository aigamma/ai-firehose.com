Of all the work AI has touched, software engineering changed first and fastest, and not by luck. Code has a property almost no other professional output shares: it is automatically and cheaply checkable. A compiler rejects what does not parse, a type checker catches whole classes of mistake, and a test suite delivers a precise verdict on whether the thing works. That single property, verifiability, is why AI for software went from autocomplete to agents that land real changes faster than almost anyone predicted.

Verifiability matters because it closes the loop a model needs to become reliable. A coding agent can generate a change, run the build, read the failure, and try again, correcting itself against a ground truth that does not flatter it, exactly the feedback that open-ended text generation never gets. Benchmarks like SWE-bench, which hand a model real issues in real repositories and grade it only on whether the tests pass, turned that loop into a public scoreboard, and the scores climbed fast precisely because the task is gradeable. Progress concentrates where it can be measured.

This reshapes what the discipline is. As models absorb more of the typing, the engineer's center of gravity moves toward specifying intent, decomposing problems, and owning correctness, the judgments a checker cannot make. But the deeper point is not that one job changed; it is that the change was legible at all. Software engineering offered AI a tight, honest feedback signal, and AI repaid it with the fastest transformation of any knowledge field, which suggests the signal, not the subject matter, was the active ingredient.

That makes software engineering the clearest preview of how AI meets any kind of work. The generalization is uncomfortable: AI helps most where the output can be checked automatically, and helps least, or harms most, where it cannot, because there the confident wrong answer goes uncaught and uncorrected. You can read off how deeply AI will reshape a field from how checkable its work is, and software, being the most checkable, simply arrived at the future first.
