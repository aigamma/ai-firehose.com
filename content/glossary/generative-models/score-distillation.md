Score distillation sampling, or SDS, borrows the knowledge inside a pretrained image diffusion model to create things the diffusion model itself cannot directly produce. Instead of sampling an image, it treats the diffusion model as a frozen critic that knows what realistic images look like, and uses that critic to optimize the parameters of some other representation. The flagship application is text-to-3D: there is far too little 3D training data to train a 3D generator the way images are trained, so SDS sidesteps the problem by optimizing a 3D scene so its rendered two-dimensional views look like good samples from a 2D text-to-image model. The 2D model supplies all the visual knowledge; the 3D structure emerges from demanding that every viewpoint be plausible.

It matters because it unlocked a whole class of generation tasks that lack their own large training sets, with text-to-3D the breakout example. The method, introduced as DreamFusion in 2022, generates a 3D asset from a text prompt with no 3D supervision at all, by repeatedly rendering a neural radiance field from random camera angles and nudging it toward what the diffusion model judges realistic for that prompt. The same idea generalizes to optimizing other parameterized objects, textures on a mesh, edits to an existing scene, or motion, anywhere a differentiable renderer can turn the thing being optimized into images a 2D diffusion prior can score.

Mechanically, SDS provides a gradient without running the diffusion sampler. At each step it renders a view of the representation, adds noise to that rendering at a random noise level, and asks the diffusion model to predict the noise; the gap between the noise the model predicts and the noise actually added points in the direction that would make the rendered view more like a real sample, and that signal is backpropagated through the renderer into the underlying parameters, the radiance field or the mesh. Crucially, the usual diffusion-training term that would differentiate through the network's own weights is dropped, the specific simplification that defines the SDS gradient and connects it to the score-based model view of diffusion as estimating the gradient of log-density. Strong classifier-free guidance, much higher than for ordinary image sampling, is needed to make the optimization commit to the prompt.

Score distillation is powerful but has well-known failure modes that frame the active research around it. Because it optimizes toward a high-probability mode rather than sampling, results can be oversaturated and oversmoothed, and the multi-view nature of the problem invites the "Janus" artifact, in which a generated object grows a plausible-from-each-angle face on multiple sides because each view independently satisfies the 2D prior without global 3D agreement. Follow-up methods refine the objective to restore diversity and detail and combine it with faster 3D representations like gaussian splatting to speed up the optimization. The enduring contribution is the recipe itself: distill a 2D generative prior into an arbitrary differentiable representation, turning an image model into a general-purpose engine for synthesizing structures it was never trained to output.
