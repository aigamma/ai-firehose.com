A vector-quantized variational autoencoder, or VQ-VAE, is an autoencoder whose latent representation is discrete instead of continuous. A standard [variational autoencoder](variational-autoencoder) encodes an input into a vector of real numbers; a VQ-VAE encodes an image into a grid of cells and snaps each cell's vector to the nearest entry in a learned codebook, a fixed dictionary of vectors. The image is therefore represented as a grid of integer indices, one per cell, naming which codebook entry each region used. The decoder reconstructs the image from those looked-up code vectors. In effect the model learns a visual vocabulary and rewrites every image as a short sentence in that vocabulary.

This matters because discreteness is what lets the powerful machinery built for language be applied to images. Once an image is a grid of tokens, an [autoregressive model](autoregressive-model), the same kind of next-token predictor used for text, can learn to generate images by predicting one code after another, and the image can be flattened into a sequence and modeled exactly as words are. This is the foundation of the token-based family of image and video generators, and the discrete codes also serve as the canonical example of [image tokenization](image-tokenization), the step that turns continuous pixels into the symbolic units a sequence model consumes. The same idea extends to audio, where a waveform is quantized into code tokens and generated autoregressively.

The quantization step introduces one technical wrinkle: snapping a continuous vector to its nearest codebook entry is a non-differentiable operation, so gradients cannot flow through it in the ordinary way. VQ-VAE handles this with a straight-through estimator, copying the gradient from the decoder's input straight back to the encoder's output as if the quantization were the identity, plus extra loss terms that pull the codebook entries toward the encoder outputs and keep the encoder committed to its chosen codes. A widely used successor, VQ-GAN, sharpens reconstructions by adding an adversarial loss from a [generative adversarial network](generative-adversarial-network) discriminator and a perceptual loss on top of the VQ-VAE backbone, so the decoded images look crisp rather than blurry at high compression.

VQ-VAE occupies a pivotal place between the major generative families. Like the autoencoder behind a [latent diffusion](latent-diffusion) model, it performs perceptual compression so that the hard generative work happens in a small space rather than on raw pixels; the difference is that latent diffusion keeps that space continuous and denoises within it, whereas a VQ-VAE makes the space discrete and lets an autoregressive prior model it as tokens. The two strategies, continuous-latent diffusion and discrete-token autoregression, are the dominant routes to high-resolution image synthesis, and several recent systems blur the boundary by running diffusion over discrete tokens or by combining token prediction with diffusion refinement.
