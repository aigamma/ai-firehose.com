A latent variable model explains the data you can see by appealing to variables you cannot. It posits that each observation was generated from some hidden quantity, the latent variable, which is never directly measured and must be inferred. The intuition is that complicated, high-dimensional data often has a simpler underlying cause: a face image is governed by a few factors like pose, lighting, and identity; a document is shaped by a handful of topics. The latent variable is that compact hidden cause, and the model specifies how it gives rise to the observation, a process called the generative story.

Latent variable models matter because they capture structure that no purely observational model can. By compressing data through a bottleneck of hidden factors, they perform dimensionality reduction, reveal clusters, separate signal from noise, and provide a generative mechanism that can synthesize new examples by sampling a latent value and running the generative story forward. Classical examples span the field: factor analysis and probabilistic PCA use continuous latents, Gaussian mixtures and topic models use discrete or categorical latents, and the [hidden Markov model](hidden-markov-model) uses latents that evolve in time. In each case the latent variables are the explanatory backbone the model is built around.

The defining technical challenge is inference: given an observation, what was the hidden variable that produced it. By [Bayes' theorem](bayes-theorem) this posterior requires the marginal likelihood of the data, an integral over all possible latent values that is usually intractable. Two families of methods answer this. [Expectation maximization](expectation-maximization) handles cases where the posterior is tractable, alternating between inferring the latents and updating the parameters. When the posterior is not tractable, [variational inference](variational-inference) approximates it by maximizing the [evidence lower bound](evidence-lower-bound), trading exactness for scalability. Learning and inference are thus inseparable in a latent variable model, because you cannot fit the parameters without reasoning about the hidden variables and vice versa.

Latent variable models are the conceptual bridge from classical probabilistic machine learning to modern deep generative models. The [variational autoencoder](variational-autoencoder) is a latent variable model whose generative story is a neural network mapping a latent code in a [latent space](latent-space) to data, with inference handled by an amortized encoder. Diffusion models can be cast as deep latent variable models with a long chain of progressively noised latents, and even a [generative adversarial network](generative-adversarial-network) shares the core move of sampling a latent and transforming it into an observation. Across all of these, the same idea recurs: posit a hidden cause, define how it generates the data, and learn to invert that generation. That pattern is one of the load-bearing ideas of the entire field.
