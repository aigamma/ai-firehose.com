Expectation maximization is an iterative recipe for finding maximum-likelihood parameters when a model has hidden, unobserved variables. Many useful models are like this: you see some data, but the quantity that would make fitting easy, such as which cluster each point belongs to, is missing. If those hidden assignments were known, estimating the parameters would be a straightforward maximum likelihood problem. The difficulty expectation maximization resolves is a chicken-and-egg one: you need the parameters to infer the hidden variables, and the hidden variables to estimate the parameters.

Hidden structure is everywhere, and expectation maximization gives a principled, monotonically improving way to recover it. The canonical example is the Gaussian mixture model, where each point is assumed to come from one of several Gaussian components but the component label is unobserved; expectation maximization recovers both the component shapes and the soft assignment of points to components, generalizing hard clustering into a probabilistic one. The same algorithm trains hidden Markov models, fills in missing data, and underlies many classical methods in speech, vision, and bioinformatics.

The algorithm alternates two steps until the parameters stop changing. In the expectation, or E, step it holds the current parameters fixed and computes the posterior distribution over the hidden variables, the model's current best belief about the missing values. In the maximization, or M, step it holds that distribution fixed and updates the parameters to maximize the expected log-likelihood under it, which for many models has a clean closed form. Each full iteration is guaranteed never to decrease the data likelihood, so the procedure climbs steadily toward a fixed point. The price of that guarantee is that the fixed point is only a local optimum, so the outcome depends on initialization and is usually run from several starting points.

The deeper view connects expectation maximization to the broader machinery of approximate inference. Both steps are coordinate ascent on a single objective, the evidence lower bound on the data log-likelihood: the E step tightens the bound by setting the auxiliary distribution to the exact posterior, and the M step raises the bound by optimizing the parameters. This is exactly the scaffolding of variational inference, and indeed expectation maximization is the special case where the posterior is tractable enough to use exactly rather than approximate. When that posterior is itself intractable, the E step is replaced by a variational or sampling approximation, which is how the same idea scales from tidy mixtures of Gaussian distributions to the large latent-variable models used in modern generative modeling.
