The Universal Approximation Theorem is the result that gives neural networks their theoretical license. In its classic form, proved for sigmoid activations by Cybenko in 1989 and extended to broad classes of activations by Hornik shortly after, it states that a feedforward network with a single hidden layer containing enough neurons can approximate any continuous function on a closed and bounded region to any desired accuracy. There is no continuous target the architecture is fundamentally unable to represent. In this precise sense a neural network is a universal approximator, which is the formal reassurance that the model family is rich enough to be worth studying.

The theorem is a statement about existence, and grasping its scope means grasping what it deliberately leaves out. It promises that suitable weights exist that make the network approximate the target, but it says nothing about how many neurons are required, which can be astronomically large, nothing about how to find those weights from data, and nothing about generalization to unseen inputs. It is an expressivity result, not a learnability result. A network can be capable in principle of representing the right function while no training procedure ever discovers it from a finite sample. Conflating universal approximation with a guarantee of practical success is a common and consequential misreading.

This is exactly where the No Free Lunch Theorem provides the counterweight. Representability does not imply learnability: among the infinitely many functions a wide network can express, the training data alone cannot single out the correct one, so an Inductive Bias is still required to choose well, and finite Sample Complexity still governs whether the right function can be identified at all. Expressive power and the ability to generalize are different axes, and a model can sit high on the first while needing strong assumptions and abundant data to make progress on the second. Universal approximation guarantees only that the target is somewhere in the haystack.

The theorem also speaks to the longstanding debate over network depth. The classic result uses a single hidden layer, so depth is not required for universality, yet a shallow network may need exponentially more neurons to match a function that a deep network represents compactly. Later depth-separation results show that some functions expressible by a deep network of modest size would demand an impractically wide shallow network. This is one theoretical reason deep architectures are favored: not that they can represent more in the limit, since both are universal, but that they often represent the functions we care about far more efficiently, with a parameter count and an Inductive Bias better suited to structured data. Universal approximation opens the door; depth, data, and the right bias are what let us walk through it.
